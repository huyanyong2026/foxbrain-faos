# Autopilot Module

FoxBrain OS Enterprise V2.2 autonomous business operation boundary.

- Page route: `/business-autopilot`
- API: `/api/v2.2`
- Storage: `business_health_scores`, `business_alerts`, `action_tasks`, `ai_learning_records`, `ceo_daily_reports`
- Rule: autopilot creates alerts, tasks and reports; high-risk execution requires human approval
