# Decision Center Module

VAFOX Enterprise OS V1.7 decision center boundary.

- Page route: `/ai-business-center`
- Primary API: `/api/decision/today`
- Data source: SAP readonly data, enterprise knowledge, operating rules and AI business memory
- Execution rule: AI can create advice and approval plans only; business execution requires human approval
- Storage: `ai_recommendation_history`, `ai_operation_plans`, `ai_tasks`
