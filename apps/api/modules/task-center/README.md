# Task Center Module

VAFOX Enterprise OS V1.8 AI task center boundary.

- API: `/api/workflow-automation/tasks`
- Storage: `tasks`, `task_logs`
- Task statuses: `todo`, `doing`, `pending_review`, `done`, `closed`
- Source examples: risk engine, AI business center, workflow engine
- Guardrail: AI-generated tasks remain reviewable and auditable.
