# Workflow Module

FoxBrain OS Enterprise V1.8 workflow automation boundary.

- Page route: `/workflow-automation`
- API: `/api/workflow-automation`
- Run API: `/api/workflow/run-inventory-risk`
- Storage: `workflows`, `workflow_nodes`, `task_logs`
- Guardrail: AI can build workflow plans and task drafts, but high-risk execution waits for human approval.
