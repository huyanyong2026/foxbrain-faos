# 权限与安全检查

## 已有控制

- 密码 PBKDF2-SHA256 加盐存储。
- 登录失败锁定。
- Cookie 为 HttpOnly、Secure、SameSite=Lax。
- HSTS、CSP、X-Frame-Options、nosniff 等响应头存在。
- 应用仅监听本机 8088，通过 Nginx 对外。
- 仓库扫描未发现明显硬编码私钥或明文凭据赋值。
- SAP 连接信息未在本报告中读取或输出。

## 风险

- P1：未发现 CSRF token，存在跨站请求风险。
- P1：会话无明确过期和服务端撤销机制。
- P1：3000、3001、5001、5678 等容器端口对公网监听，需确认安全组、认证和用途。
- P1：SQLite 外键关闭，权限关联和对象关系容易产生不一致。
- P2：CSP 允许 `unsafe-inline`。
- P2：账号模型当前仅 admin 与 disabled employee，完整岗位权限矩阵未通过真实账号验收。
- P2：新版 Drive 权限表存在，但 AI 检索权限继承无生产处理记录可证明。
- P2：生产日志含 Traceback，应确认是否包含用户数据或连接信息。
