# 前端检查

## 已确认

- 当前一级导航已统一为五项：首页、经营、企业档案、AI助手、行动中心。
- 主要页面具备中文化、空状态、证据卡、全局 Copilot 和手机布局代码。
- 安全响应头包含 CSP、X-Frame-Options、nosniff、HSTS、Referrer-Policy。
- 未登录访问业务页面会跳转登录，公开健康接口可访问。
- 本机只读 GET 响应约 1-4 毫秒，不代表外网完整页面性能。

## 问题

- 仍有大量隐藏旧页面，用户从历史链接仍可能进入旧体验。
- 单文件内存在乱码文案片段，说明历史字符编码治理未完全结束。
- CSP 允许 inline script 和 inline style，降低 XSS 防护强度。
- 未进行登录后全路径点击测试，因为无法在不使用真实凭据的情况下完成。
- 失效按钮、空页面和浏览器端交互未能全量确认。
