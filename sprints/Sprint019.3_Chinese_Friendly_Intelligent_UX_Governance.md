# Sprint019.3：全中文友好智能体验与无效页面治理

状态：准备执行  
目标站点：huyan.vafox.com  
优先级：P0  
依赖：Sprint019.1、Sprint019.2

---

## 一、目标

把 FoxBrain 从“技术系统界面”升级为“全中文、友好、流畅、智能、老板看得懂”的企业工作台。

原则：

- 除品牌正式名称外，所有用户可见页面必须使用中文。
- 不向普通用户展示英文模块名、英文状态、数据库字段、接口名称、代码、路径、异常堆栈。
- 不保留无效入口、空白页面、半成品页面、无法操作的按钮。
- 每一个页面必须让用户知道：这里是什么、当前有什么、下一步做什么。
- 技术能力保留在后台，但前台必须用经营语言表达。

---

## 二、全中文治理

全面扫描并替换以下内容：

- 页面标题
- 一级和二级导航
- 卡片标题
- 按钮
- 表格字段
- 状态标签
- 空状态
- 错误提示
- 加载提示
- 搜索提示
- Copilot 回答结构
- 文件处理状态
- 数据同步状态
- 系统通知

允许保留英文的范围仅限：

- 正式品牌名，例如 Kailas、Osprey、Mammut、VAFOX
- 用户上传文件中的原文
- 管理员专用技术详情页中主动展开后的原始信息

必须替换示例：

- Dashboard → 经营首页
- Copilot → AI助手
- Decision Engine → 经营决策
- Business Health → 企业健康
- Daily Intelligence → 每日经营简报
- Sync Center → 数据同步中心
- Data Lake → 企业数据湖
- Knowledge Graph → 企业关系图谱
- Evidence → 依据
- Rebuild → 重新分析
- Pending → 待处理
- Success → 已完成
- Failed → 失败
- Empty / No Data → 暂无数据，并说明原因与下一步

---

## 三、技术内容隐藏

普通用户页面禁止直接显示：

- API 路径
- 数据库表名
- Python、SQL、JSON、HTML、代码块
- Commit、Branch、SHA
- 服务器目录
- 系统堆栈
- 内部字段名
- 英文枚举值

发生异常时，前台只显示：

- 发生了什么
- 是否影响数据
- 用户现在可以做什么
- 是否需要联系管理员

技术详情必须折叠在“管理员诊断信息”中，默认隐藏。

---

## 四、无效页面治理

全面盘点所有路由和菜单，分类：

1. 正常可用：保留
2. 数据未准备：显示友好引导
3. 功能未完成：从导航隐藏
4. 重复页面：合并
5. 无入口孤立页面：删除或归入系统管理
6. 404/500/空白页面：全部修复

任何页面不得只显示：

- 空白
- 原始 JSON
- “No data”
- 英文报错
- 无法点击的按钮
- 没有说明的空表格

友好空状态必须包含：

- 当前为什么没有内容
- 需要什么数据
- 可执行按钮
- 可直接询问 AI

示例：

“当前还没有南山店库存分析。请先同步最新库存数据，或点击‘重新分析’。也可以直接询问 AI：南山店目前缺少哪些数据？”

## 五、统一页面结构

所有主要页面统一为：

1. 中文页面标题
2. 一句经营说明
3. 当前状态
4. 核心数字
5. 风险或机会
6. 建议行动
7. 依据
8. 数据更新时间
9. AI提问入口

不再让用户理解系统架构或引擎名称。

## 七、AI助手中文化

AI助手所有回答必须使用中文，并统一结构：

- 结论
- 为什么
- 关键数字
- 风险与不确定性
- 建议行动
- 依据
- 数据更新时间

禁止出现：

- 英文模板标题
- 原始字段名
- 代码
- 内部表名
- 无法解释的技术术语

若数据不足，必须说明：

“当前数据不足，暂时无法形成可靠结论。”

并同时说明：

- 缺少什么
- 如何补齐
- 可以点击什么

## 八、友好流畅体验

必须实现：

- 页面加载骨架
- 清晰加载状态
- 操作成功提示
- 失败后可重试
- 防重复提交
- 长任务显示进度
- 手机端可操作
- 表格支持搜索、筛选、分页
- 返回路径明确
- 页面切换不丢失用户位置
- 重要按钮固定在容易找到的位置

性能要求：

- 首页首屏尽快显示核心信息
- 慢数据分区异步加载
- 不因为一个模块失败导致整页失败
- 无效请求和重复查询必须减少

## 十一、验收标准

- 用户可见页面除品牌名外不再出现英文模块名。
- 不显示代码、接口、表名、字段名、堆栈。

- 首页、经营、企业档案、AI助手、行动中心手机端可正常使用。
- AI回答完全中文化。
- 企业网盘全部中文化并可在线打开文件。
- 页面响应和操作流畅度明显提升。
- 不增加 SAP 写权限。
- 不修改生产 SAP。
- 不开发 ai.vafox.com。
