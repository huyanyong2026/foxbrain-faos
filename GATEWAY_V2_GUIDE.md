# Gateway V2 Guide — VAFOX Portal V2

## Purpose
Gateway V2 is the unified enterprise portal for login, identity, permission, smart routing, and the role dashboard.

## User Roles
Supported roles include CEO, employee, store manager, procurement, supplier, and customer. Existing authentication and approval workflows remain unchanged.

## Smart Routing
After login, the Portal V2 surface maps users to their operating destination:
- CEO/admin → Huyan V2 CEO Operating System.
- Employee/supplier/customer → AI Workforce OS V2.
- Procurement → Supply Chain Workspace.
- Store Manager → Store Intelligence.

## Portal Dashboard
The dashboard shows My Role, My AI Assistant, My Tasks, My Notifications, and System Status.

## Route
Open `/experience` or `/gateway-v2` after login.
