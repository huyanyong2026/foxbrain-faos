# AI OS V5.1 Architecture

FoxBrain AI OS V5.1 connects Identity → Data → AI → Decision → Action → Memory.

## Systems

- Gateway: identity, permission, and route control.
- Huyan: CEO Autonomous Command Center.
- AI Workspace: natural-language AI Router and agent orchestration.
- Core: Enterprise Digital Twin and read-only business data context.

## Acceptance

All services must run `AI-OS-V5.1`; mixed versions are blocked before deployment.
