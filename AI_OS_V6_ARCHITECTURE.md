# FoxBrain AI OS V6 Architecture

Version: AI-OS-V6-CLEAN-REBUILD-V1

## Principle
FoxBrain V6 rebuilds the platform as a clean autonomous AI enterprise operating system. Legacy UI, manual workflows, broken links, and mixed runtime versions are not accepted as execution dependencies.

## Target System Flow
Users enter through `gateway.vafox.com`. Identity Intelligence sends them to the AI Router, then to `huyan.vafox.com` for the CEO Autonomous OS or `ai.vafox.com` for the Digital Workforce OS. Both consume `core.vafox.com`, the Enterprise Digital Twin, which preserves SAP B1 as the business truth layer.

## Runtime Layers
1. Gateway V6: Intelligent Enterprise Router.
2. Huyan V6: CEO Autonomous Command Center.
3. AI Workspace V6: Universal AI Conversation Interface.
4. Core V6: Enterprise Digital Twin.
5. Automation Control Center: data, AI, link, version, and workflow health.
6. Release Guard: blocks mixed frontend, backend, API, database, or route versions.

## Data Flow
SAP B1 → Core → AI → Decision → Action → Learning.
