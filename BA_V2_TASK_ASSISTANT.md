# BA V2.0-C Task Assistant

## Goal
Transform AI from answering into working.

## Workflow
AI Signal → Task Creation Draft → Human Approval → Assignment → Tracking → Completion Feedback.

## Supported Actions
Task Creation, Task Assignment, Task Tracking, and Task Completion Feedback.

## Example
Inventory issue detected → create `Check Nanshan inventory` → assign Store Manager → track completion.

## Control
Tasks created by AI start as approval-gated drafts and retain evidence links.
