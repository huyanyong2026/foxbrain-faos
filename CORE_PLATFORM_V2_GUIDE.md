# Core Platform V2 Guide — Enterprise Data Platform UI

## Purpose
Core Platform V2 provides the enterprise data management interface while preserving the existing Core V2 read-only data foundation.

## Modules
- Master Data: product, brand, store, supplier, customer, and employee.
- Event Center: sales, inventory, purchase, customer, and task events.
- Data Governance: data quality, data source, data status, and audit.
- API Center: available APIs, status, usage, and permission.

## Guardrails
Core Platform V2 does not create a duplicate data system and does not mutate SAP business logic. It displays governed data and links users to existing data-lake/system areas.

## Route
Open `/core-v2` or `/data-platform-v2` after login.
