# FoxBrain Enterprise OS V1.6

## Enterprise Knowledge Graph（企业知识网络）建设计划

目标：让企业对象、资料、经营数据、决策记录形成关联网络。

## 核心能力

1. 对象关系网络
- 品牌
- 门店
- 产品
- 员工
- 客户
- 供应商
- 合同
- 决策

2. 资料关联
文件不再孤立，自动关联对应企业对象。

3. 经营关系
关联 SAP 销售、库存、利润与企业对象。

4. CEO 查询
支持自然语言查询企业关系。

示例：

- 为什么这个品牌利润下降？
- 哪些供应商影响库存风险？
- 哪些门店需要重点关注？

5. AI 规则
必须 Evidence First。
数据不足时明确说明。

## 边界

- SAP 保持独立
- 不开发 ai.vafox.com 页面
- 不增加 SAP 写权限
- 继续优先建设 huyan.vafox.com
