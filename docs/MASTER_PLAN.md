# FoxBrain Enterprise OS Master Plan

Version: Alpha 0.1
Target Domain: huyan.vafox.com
Owner: Huohu Fox / 火狐狸

## Mission

FoxBrain exists to become the enterprise second brain of Huohu Fox.

It must preserve enterprise knowledge, connect enterprise data, understand enterprise relationships, remember decisions, and assist the CEO in making better decisions.

## Product Boundary

### In Scope Now

- huyan.vafox.com
- CEO Brain
- FoxBrain Drive
- Object Engine
- Knowledge Pipeline
- Global Search
- Timeline
- CEO Dashboard
- SAP data import / sync foundation

### Out of Scope Now

- ai.vafox.com employee platform
- public SaaS platform
- external customer portal
- supplier portal
- mobile app

These will be planned later after CEO Brain is stable.

## Core Philosophy

- One Enterprise, One Brain
- Everything is an Object
- Every File Has Value
- AI First
- Data Belongs to the Enterprise

## Definition of Done

Every sprint is complete only when product requirements are implemented, UI is usable, backend API works, database migration is stable, errors are handled, existing functions are not broken, tests or manual checklist are completed, deployment succeeds, rollback path exists, and README or sprint document is updated.
