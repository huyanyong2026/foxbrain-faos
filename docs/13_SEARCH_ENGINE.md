# 13 Search Engine / 搜索中心

## 搜索范围

VAFOX 搜索中心需要覆盖：

- 统一档案
- 知识条目
- 文件中心
- 文档切片
- SAP B1 摘要数据
- 任务和工作流
- 操作日志
- AI 查询历史

## 当前版本

Task003 使用 SQLite `LIKE` 做安全的基础搜索，覆盖知识标题、正文、摘要、关键词和标签。搜索结果以卡片优先，避免密集 ERP 表格。

## 权限过滤

搜索必须先经过登录和权限判断：

- `public_internal`: 内部用户可见
- `manager_only`: 老板、管理员、店长可见
- `finance_only`: 老板、管理员、财务可见
- `owner_only`: 创建人可见
- `restricted`: 老板、管理员可见

## AI 查询关系

AI 查询中心会先调用知识搜索，返回引用准备结构。当前阶段不直接生成未经验证的经营答案，只列出相关知识来源和限制说明。
