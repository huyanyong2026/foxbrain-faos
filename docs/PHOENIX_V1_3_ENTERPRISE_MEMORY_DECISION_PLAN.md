# FoxBrain Phoenix V1.3 Enterprise Memory & Decision Plan

## 目标

继续打磨 huyan.vafox.com，使其从经营分析工具升级为企业经营记忆与决策助手。

## 原则

- 不增加大型模块
- 不开发 ai.vafox.com
- 不修改生产 SAP
- 所有建议必须有依据

## 重点建设

### 1. 企业记忆

- 决策历史沉淀
- 经营事件记录
- 重要文件关联
- AI 自动生成企业经验

### 2. 决策中心

统一展示：

结论

依据

风险

建议

行动

结果反馈

### 3. 企业时间轴

串联：

销售变化
库存变化
品牌变化
门店变化
AI建议
经营决策

### 4. Copilot增强

支持基于企业记忆、历史决策和经营数据回答问题。

### 5. 中文体验继续治理

隐藏技术信息，让老板看到经营内容。

## 验收标准

- 老板可以快速理解企业变化
- AI回答可追溯
- 决策有历史依据
- 页面简单自然
