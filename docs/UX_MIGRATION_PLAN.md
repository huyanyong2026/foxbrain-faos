# UX_MIGRATION_PLAN

VAFOX UX-V2.0 upgrades the user experience layer only. It reuses gateway.vafox.com, huyan.vafox.com, ai.vafox.com and core.vafox.com without rebuilding backend foundations, changing SAP logic or creating duplicate data sources.

## Scope
- AI understands business signals from existing Core and AI Platform read models.
- AI discovers issues, creates tasks, waits for human decisions and records learning.
- RBAC, ABAC and audit log remain unchanged.

## Acceptance
- Huyan: CEO sees enterprise health, risks, opportunities and decision support immediately.
- AI Workforce: users see AI Team, Tasks, Knowledge and Memory.
- Gateway: role-based smart routing to Huyan, AI Workspace, Supply Chain, Store Intelligence and Core.
- Core: data platform UI focuses on Master Data, Events, Data Quality and API Status.
