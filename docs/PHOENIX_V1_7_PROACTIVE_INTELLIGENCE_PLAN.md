# FoxBrain Enterprise OS V1.7

## 主动经营智能阶段

目标：让 FoxBrain 从回答问题升级为主动发现经营风险和机会。

## 建设重点

1. 主动风险发现
- 库存风险
- 毛利变化
- 销售异常
- 合同到期
- 经营指标异常

2. 主动机会发现
- 畅销产品机会
- 品牌增长机会
- 门店提升机会
- 客户运营机会

3. 企业智能规则
- 基于 SAP 经营事实
- 基于企业资料
- 基于企业记忆
- 基于历史决策

4. CEO 智能日报升级
输出：
- 今天发生什么
- 为什么发生
- 风险在哪里
- 建议做什么

## 边界

- SAP 保持独立
- 不修改生产 SAP
- 不增加 SAP 写权限
- 不开发 ai.vafox.com 页面
- 继续优先建设 huyan.vafox.com
