# 43 Recruitment Engine

## Goal

Recruitment Engine keeps candidate tracking simple and manager-friendly.

## Model

`hr_candidates`

- Candidate name
- Phone
- Target position
- Resume file reference
- Interview status
- Evaluation
- Next step

## Workflow

1. Add candidate.
2. Upload or reference resume.
3. Record interview notes.
4. Set next step.
5. Create onboarding task after approval.

## Safety

Candidate data is sensitive and should only be visible to authorized managers.
