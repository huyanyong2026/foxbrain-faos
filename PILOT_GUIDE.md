# Pilot Guide

1. Deploy the API Gateway and only then the AI Workspace and Huyan applications.
2. Set `NEXT_PUBLIC_API_BASE_URL` to the gateway for the target environment.
3. Verify authenticated, read-only dashboard and workspace requests.
4. Confirm staff understand that AI outputs are recommendations requiring human review.
5. Record rollback owner, build tag, and validation evidence before enabling pilot users.
