# FoxBrain / 火狐狸 AI 企业经营系统

FoxBrain 是火狐狸内部 AI 企业经营系统的第一版代码骨架，定位为统一入口：

- AI 总经理
- 经营总览
- 门店中心
- 员工中心
- 品牌中心
- 产品中心
- 供应商中心
- 顾客/会员中心
- 财务中心
- 库存采购中心
- 内容发布中心
- 知识中心
- AI 智能体查询
- 任务中心
- 系统管理

## 文件说明

- `portal_v2.py`：员工登录门户、角色权限、经营首页、档案模块、知识库入口。
- `sync_sap_b1.py`：SAP B1 SQL Server 到本地 PostgreSQL 的同步脚本，并生成经营摘要。
- `sap_b1_sync_page.md`：SAP B1 同步说明页面。
- `wiki_ai_plan_content.json`：Wiki 知识库初始化内容。
- `.env.example`：环境变量模板，不包含真实密码。

## 启动方式

```bash
python3 portal_v2.py
```

默认监听：

```text
127.0.0.1:8088
```

生产环境建议使用 Nginx/Caddy 做 HTTPS 反向代理。

## 安全说明

不要把真实密码、数据库连接、API Key 写入代码或提交到 GitHub。

请复制 `.env.example` 为 `.env`，再填写真实配置。
